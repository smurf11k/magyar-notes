# The Hungarian Alphabet and Pronunciation

## About the Hungarian Language

The Hungarian language (magyar nyelv) is a unique language that is not similar to most European languages. It belongs to the Finno-Ugric language family, so it has nothing in common with the neighboring Slavic or Germanic languages.

::info

**Interesting fact**:

Hungarian is most closely related to Finnish and Estonian, although they are very different.

::

---

## The Hungarian Alphabet

The Hungarian language has **44 letters** (including digraphs — letters consisting of two characters, and one trigraph — a letter consisting of three characters).

::note

All digraphs and the one trigraph are consonants.

::

## Characteristics of the Hungarian Alphabet

### Long and Short Vowels Are Different Letters

| Short | Long |
| :---- | :--- |
| a     | á    |
| e     | é    |
| i     | í    |
| o     | ó    |
| ö     | ő    |
| u     | ú    |
| ü     | ű    |

::warning

Changing the length of a vowel **changes the meaning of the word**.

For example: **ujj** (finger) and **új** (new).

::

---

## Digraphs (2 letters = 1 sound)

| Digraph | Pronunciation                   |
| :------ | :------------------------------ |
| cs      | [ch] as in '**ch**urch'         |
| sz      | [s] as in '**s**un'             |
| zs      | [zh] as in 'vi**si**on'         |
| gy      | [dy] (soft 'd' like '**du**ty') |
| ty      | [ty] (soft 't' like '**tu**ne') |
| ny      | [ny] as in 'ca**ny**on'         |
| dz      | [dz] as in 'goo**ds**'          |
| dzs     | [j] as in '**j**ungle'          |

## Stress

In Hungarian, the stress ALWAYS falls on the **first syllable**.

| Word     | Pronunciation                             |
| :------- | :---------------------------------------- |
| szépen   | <SpeakButton text="szépen" />             |
| köszönöm | <SpeakButton text="köszönöm" />           |
| magyar   | <SpeakButton text="magyar"/>              |
| Budapest | <SpeakButton text="budapest" onlyLocal /> |

**No exceptions**:

Even in very long words.

For example: [**összehasonlíthatatlanul**](https://forvo.com/search/%C3%B6sszehasonl%C3%ADthatatlanul/)

## Tasks

::steps

### Learn the alphabet

- Pronunciation of all letters
- Repeat each letter out loud 5 times
- Pay special attention to: á, é, ö, ő, ü, ű

### Digraphs

- Learn all digraphs (cs, sz, zs, gy, ty, ny, dz, dzs)
- Practice pronouncing: cs-sz, s-sz, z-zs

### First words

- Learn 10 simple words
- You can record yourself on a voice recorder and compare with the correct pronunciation
- Repeat until it sounds similar

### Stress

- Take any 10 Hungarian words
- Identify the first syllable
- Pronounce with the correct stress
