# Множина іменників

## Основне правило

Множина в угорській мові утворюється додаванням закінчення **-k** до іменника, але перед ним часто з'являється сполучна голосна.

### Варіанти закінчення:

- **-k** (після голосної)
- **-ok/-ek/-ök** (після приголосної)

::info

**Голосна гармонія** (як завжди):

- **Задні голосні**: **-ok**
- **Передні неокруглені**: **-ek**
- **Передні округлені**: **-ök**

::

---

## Приклади утворення множини

### Слова на голосну (просто додаємо -k):

| Однина | Множина | Переклад         |
| ------ | ------- | ---------------- |
| alma   | almák   | яблуко → яблука  |
| autó   | autók   | машина → машини  |
| ceruza | ceruzák | олівець → олівці |
| kávé   | kávék   | кава → кави      |

### Слова на приголосну (додаємо -ok/-ek/-ök):

| Однина | Множина  | Переклад            |
| ------ | -------- | ------------------- |
| ház    | házak    | будинок → будинки   |
| könyv  | könyvek  | книга → книги       |
| ember  | emberek  | людина → люди       |
| tükör  | tükrök   | дзеркало → дзеркала |
| gyerek | gyerekek | дитина → діти       |

### Приклади в реченнях:

- **Látok három almát.** <SpeakButton text="Látok három almát." label="" /> - Я бачу три яблука.
- **Olvasok könyveket.** <SpeakButton text="Olvasok könyveket." label="" /> - Я читаю книги.
- **Vannak gyerekek.** <SpeakButton text="Vannak gyerekek." label="" /> - Є діти.

## Особливі випадки

### 1. Скорочення довгих голосних:

Деякі слова скорочують довгу голосну перед множиною:

| Однина | Множина | Переклад        |
| ------ | ------- | --------------- |
| út     | utak    | дорога → дороги |
| híd    | hidak   | міст → мости    |
| ág     | ágak    | гілка → гілки   |
| tó     | tavak   | озеро → озера   |

### 2. Подовження коротких голосних:

Деякі слова подовжують короткі голосні:

| Однина | Множина | Переклад      |
| ------ | ------- | ------------- |
| nyár   | nyarak  | літо → літа   |
| tél    | telek   | зима → зими   |
| szó    | szavak  | слово → слова |

### 3. Зміна основи:

Деякі слова повністю змінюють основу:

| Однина | Множина | Переклад      |
| ------ | ------- | ------------- |
| ló     | lovak   | кінь → коні   |
| tó     | tavak   | озеро → озера |

::warning

**Увага**: Ці нерегулярні форми треба запам'ятовувати окремо!

::

## Множина + Знахідний відмінок

Коли множина поєднується зі знахідним відмінком, спочатку додається **-k** (множина), потім **-t** (знахідний):

### Формула: іменник + **-k** + **-at/-et/-öt**

| Однина | Множина | Множ. + знахідний | Переклад                    |
| ------ | ------- | ----------------- | --------------------------- |
| alma   | almák   | almákat           | яблуко → яблука (знахідний) |
| könyv  | könyvek | könyveket         | книга → книги (знахідний)   |
| autó   | autók   | autókat           | машина → машини (знахідний) |
| ember  | emberek | embereket         | людина → людей              |

### Приклади:

- **Látok almákat.** <SpeakButton text="Látok almákat." label="" /> - Я бачу яблука.
- **Olvasok könyveket.** <SpeakButton text="Olvasok könyveket." label="" /> - Я читаю книги.
- **Szeretem az autókat.** <SpeakButton text="Szeretem az autókat." label="" /> - Я люблю машини.
- **Ismerem az embereket.** <SpeakButton text="Ismerem az embereket." label="" /> - Я знаю цих людей.

## Числівники та множина

::note

**Важливо**: Після числівників іменник стоїть в **ОДНИНІ**!

✅ **három alma** (три яблука - однина!)  
❌ **három almák** (неправильно)

✅ **öt könyv** (п'ять книг - однина!)  
❌ **öt könyvek** (неправильно)

::

### Але зі знахідним:

- **Látok három almát.** <SpeakButton text="Látok három almát." label="" /> - Я бачу три яблука (однина + знахідний).
- **Olvasok öt könyvet.** <SpeakButton text="Olvasok öt könyvet." label="" /> - Я читаю п'ять книг (однина + знахідний).

### Без числівника - множина:

- **Látok almákat.** <SpeakButton text="Látok almákat." label="" /> - Я бачу яблука (множина + знахідний).
- **Olvasok könyveket.** <SpeakButton text="Olvasok könyveket." label="" /> - Я читаю книги (множина + знахідний).

## Ванк (Van/Vannak)

Дієслово "бути" має дві форми:

- **van** - для однини: **Van egy alma.** (Є яблуко)
- **vannak** - для множини: **Vannak almák.** (Є яблука)

### Приклади:

- **Van egy könyv az asztalon.** <SpeakButton text="Van egy könyv az asztalon." label="" /> - На столі є книга.
- **Vannak könyvek az asztalon.** <SpeakButton text="Vannak könyvek az asztalon." label="" /> - На столі є книги.

### З запереченням:

- **Nincs alma.** <SpeakButton text="Nincs alma." label="" /> - Немає яблука (однина).
- **Nincsenek almák.** <SpeakButton text="Nincsenek almák." label="" /> - Немає яблук (множина).

## Діалоги

### Діалог 1: У магазині

**A:** Mit keresel?  
**B:** Keresek almákat és banánokat.  
**A:** Hány almát?  
**B:** Öt almát kérek.  
**A:** És hány banánt?  
**B:** Három banánt.

### Діалог 2: У бібліотеці

**A:** Vannak magyar könyvek?  
**B:** Igen, vannak. Hány könyvet szeretnél?  
**A:** Két könyvet kérek.  
**B:** Tessék, ezek jó könyvek.

## Завдання

::steps

### Утворення множини

- Утворити множину від 30 різних іменників
- Практикувати нерегулярні форми (ló → lovak, tó → tavak)
- Вивчити слова з подовженням/скороченням голосних

### Множина + знахідний

- Утворити форму множина + знахідний від 20 слів
- Скласти 10 речень з множиною у знахідному відмінку
- Практикувати різницю між "három alma" та "almákat"

### Практика з числівниками

- Написати 10 речень з числівниками (однина після числівника!)
- Написати 10 речень без числівників (множина)
- Перекласти речення з української на угорську

::
