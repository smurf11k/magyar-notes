# Члени сім'ї

## Найближча родина

### Батьки та діти:

| Українською | Угорською    | Вимова         |
| ----------- | ------------ | -------------- |
| мама        | anya         | [онья]         |
| тато        | apa          | [опо]          |
| батьки      | szülők       | [сюлек]        |
| син         | fiú / fia    | [фіу / фіо]    |
| дочка       | lány / lánya | [лань / ланьо] |
| діти        | gyerekek     | [дьерекек]     |
| дитина      | gyerek       | [дьерек]       |
| немовля     | csecsemő     | [чечемє]       |
| малюк       | kisbaba      | [кішбобо]      |

::info

**Fiú vs Fia, Lány vs Lánya**:

- **fiú** / **lány** - хлопчик / дівчинка (загалом)
- **fia** / **lánya** - його/її син / його/її дочка (з присвійним закінченням)

Приклади:

- **Ez egy fiú.** - Це хлопчик.
- **Ez a fiam.** - Це мій син.
- **Ez egy lány.** - Це дівчинка.
- **Ez a lányom.** - Це моя дочка.

::

## Брати та сестри

| Українською     | Угорською                 | Вимова                      |
| --------------- | ------------------------- | --------------------------- |
| брат            | fiútestvér / öcs / báty   | [фіутештвер / еч / бать]    |
| сестра          | lánytestvér / húg / nővér | [ланьтештвер / хуг / нєвер] |
| молодший брат   | öcs                       | [еч]                        |
| старший брат    | báty                      | [бать]                      |
| молодша сестра  | húg                       | [хуг]                       |
| старша сестра   | nővér                     | [нєвер]                     |
| брати та сестри | testvérek                 | [тештверек]                 |
| близнюки        | ikrek                     | [ікрек]                     |
| близнюк (один)  | iker                      | [ікер]                      |

::warning

**Важливо про братів та сестер**:

В угорській мові є окремі слова залежно від того, старший чи молодший брат/сестра:

- **öcs** - молодший брат (від мене)
- **báty** - старший брат (від мене)
- **húg** - молодша сестра (від мене)
- **nővér** - старша сестра (від мене)

Якщо не хочете уточнювати:

- **fiútestvér** - брат (нейтрально)
- **lánytestvér** - сестра (нейтрально)
- **testvér** - брат/сестра (загалом)

::

## Бабусі, дідусі та онуки

| Українською       | Угорською                  | Вимова                       |
| ----------------- | -------------------------- | ---------------------------- |
| бабуся            | nagymama / nagyi           | [нодьмомо / нодьї]           |
| дідусь            | nagypapa / nagyapa / nagyi | [нодьпопо / нодьопо / нодьї] |
| бабуся та дідусь  | nagyszülők                 | [нодьсюлек]                  |
| онук              | unoka (fiú)                | [уноко]                      |
| онучка            | unoka (lány)               | [уноко]                      |
| онуки             | unokák                     | [унокак]                     |
| правнук/правнучка | dédunoka                   | [дедуноко]                   |

::tip

**Пестливі форми**:

- **nagymama** → **mamó** [момо] - бабуся (пестливо)
- **nagypapa** → **papó** [попо] - дідусь (пестливо)
- **nagyi** [нодьї] - баба/дід (дуже розмовно, для обох)

::

## Тітки, дядьки та їхні діти

| Українською            | Угорською           | Вимова           |
| ---------------------- | ------------------- | ---------------- |
| тітка                  | nagynéni            | [нодьнені]       |
| дядько                 | nagybácsi           | [нодьбачі]       |
| двоюрідний брат        | unokatestvér (fiú)  | [унокотештвер]   |
| двоюрідна сестра       | unokatestvér (lány) | [унокотештвер]   |
| двоюрідні брати/сестри | unokatestvérek      | [унокотештверек] |
| племінник              | unokaöcs            | [унокоеч]        |
| племінниця             | unokahúg            | [унокохуг]       |

::note

**Слово "unoka"**:

Слово **unoka** дуже багатозначне:

- unoka - онук/онучка
- unokatestvér - двоюрідний брат/сестра
- unokaöcs - племінник (молодший)
- unokahúg - племінниця (молодша)

Це все пов'язано з ідеєю "покоління далі" від прямої лінії.

## Як говорити про сім'ю

### Базові конструкції:

**1. У мене є...**

- **Van egy bátyám.** - У мене є старший брат.
- **Van két húgom.** - У мене є дві молодші сестри.
- **Vannak testvéreim.** - У мене є брати/сестри.
- **Nincs testvérem.** - У мене немає братів/сестер.

**2. Моя сім'я...**

- **A családom nagy.** - Моя сім'я велика.
- **A családom kicsi.** - Моя сім'я маленька.
- **Négy fős a családunk.** - Наша сім'я з чотирьох осіб.

**3. Члени сім'ї + прикметники:**

- **Az anyám kedves.** - Моя мама мила.
- **Az apám magas.** - Мій тато високий.
- **A bátyám sportos.** - Мій старший брат спортивний.
- **A húgom okos.** - Моя молодша сестра розумна.

## Присвійні закінчення (вступ)

Щоб сказати "мій тато", "твоя мама" тощо, в угорській додаються спеціальні закінчення:

### З словом "anya" (мама):

| Особа | Форма   | Переклад     |
| ----- | ------- | ------------ |
| én    | anyám   | моя мама     |
| te    | anyád   | твоя мама    |
| ő     | anyja   | його/її мама |
| mi    | anyánk  | наша мама    |
| ti    | anyátok | ваша мама    |
| ők    | anyjuk  | їхня мама    |

### З словом "apa" (тато):

| Особа | Форма  | Переклад     |
| ----- | ------ | ------------ |
| én    | apám   | мій тато     |
| te    | apád   | твій тато    |
| ő     | apja   | його/її тато |
| mi    | apánk  | наш тато     |
| ti    | apátok | ваш тато     |
| ők    | apjuk  | їхній тато   |

::warning

**Це лише ознайомлення!**:

Ми детально вивчимо присвійні закінчення в уроці 19. Зараз просто запам'ятайте найпоширеніші форми:

- **-m** (мій): anyám, apám, bátyám
- **-d** (твій): anyád, apád, bátyád
- **-ja/-je** (його/її): anyja, apja, bátyja

::

## Корисні фрази про сім'ю

### Запитання:

- **Van testvéred?** - У тебе є брати/сестри?
- **Hány testvéred van?** - Скільки у тебе братів/сестер?
- **Van családod?** - У тебе є сім'я?
- **Hányan vagytok a családban?** - Скільки вас у сім'ї?
- **Milyen a családod?** - Яка твоя сім'я?

### Відповіді:

- **Van egy bátyám és egy húgom.** - У мене є старший брат і молодша сестра.
- **Négy fős a családunk.** - Нас четверо в сім'ї.
- **Egyke vagyok.** - Я єдина дитина.
- **Hárman vagyunk testvérek.** - Нас троє братів/сестер.
- **Nagy család vagyunk.** - Ми велика сім'я.

::tip

**Слово "egyke"**:

**Egyke** [еджке] - єдина дитина (без братів та сестер).

- **Egyke vagyok.** - Я єдина дитина.
- **Ő egyke.** - Він/вона єдина дитина.

::

## Діалог: Розповідь про сім'ю

### Діалог 1 (неформальний):

**Anna:** Van testvéred?  
**Péter:** Igen, van egy bátyám. És neked?  
**Anna:** Nekem van két húgom. Ikrek.  
**Péter:** Tényleg? Hány évesek?  
**Anna:** Tizenhat évesek. És a bátyád?  
**Péter:** Ő huszonnyolc éves. Már van családja is.  
**Anna:** Van gyereke?  
**Péter:** Igen, két gyereke van. Egy fiú és egy lány.  
**Anna:** Szuper! Akkor te már nagybácsi vagy!  
**Péter:** Igen, nagyon szeretem az unokaöcsémet és az unokahúgomat.

### Діалог 2 (формальний):

**Kovács úr:** Milyen a családja?  
**Nagy úr:** Négy fős a családunk. A feleségem, én és a két gyerekünk.  
**Kovács úr:** Hány évesek a gyerekei?  
**Nagy úr:** A fiam tizenkét éves, a lányom kilenc.  
**Kovács úr:** Szépen neveltek! Vannak testvérei?  
**Nagy úr:** Igen, van egy húgom. Ő Pécsett lakik.  
**Kovács úr:** És a szülei?  
**Nagy úr:** A szüleim már nyugdíjasok. Budapesten élnek.

## Опис членів сім'ї

### Корисні прикметники для опису людей:

**Зовнішність:**

- **magas** - високий
- **alacsony** - низький
- **karcsú** - стрункий
- **kövér** - товстий
- **szőke** - білявий
- **barna** - темноволосий/шатен
- **vörös** - рудий
- **ősz** - сивий

**Характер:**

- **kedves** - милий
- **aranyos** - симпатичний
- **okos** - розумний
- **vicces** - веселий
- **szigorú** - суворий
- **türelmes** - терплячий
- **gondoskodó** - турботливий
- **segítőkész** - допомагаючий

### Приклади речень:

- **Az anyám magas és karcsú.** - Моя мама висока і струнка.
- **Az apám ősz hajú és kedves.** - Мій тато сивий і милий.
- **A húgom vicces és okos.** - Моя молодша сестра весела і розумна.
- **A nagypapám szigorú, de gondoskodó.** - Мій дідусь суворий, але турботливий.

## Завдання

### Завдання 1: Лексика

- Вивчити всі назви найближчих родичів (anya, apa, testvér...)
- Вивчити слова для братів/сестер (öcs, báty, húg, nővér)
- Вивчити слова для бабусь/дідусів та онуків
- Вивчити слова для родичів через шлюб

### Завдання 2: Присвійні форми

- Вивчити форми: anyám, apám, bátyám, húgom
- Вивчити форми: anyád, apád, bátyád, húgod
- Практикувати з 5 різними словами сімейних членів

### Завдання 3: Розповідь про сім'ю

- Написати текст про свою сім'ю (10-15 речень)
- Включити: скільки вас у сім'ї, хто є в сім'ї, вік членів сім'ї
- Описати 2-3 членів сім'ї (зовнішність + характер)
- Прочитати текст вголос та записати

### Завдання 4: Діалоги

- Скласти діалог "Знайомство з сім'єю друга"
- Скласти діалог "Розповідь про батьків"
- Скласти діалог "Про братів та сестер"
- Практикувати кожен діалог мінімум 5 разів

Створіть своє сімейне древо угорською! Включіть:

   - nagyanyám (бабуся по материнській лінії)
   - nagyapám (дідусь по материнській лінії)
   - nagymamám (бабуся по батьківській лінії)
   - nagypapám (дідусь по батьківській лінії)

2. **Szülők** (Батьки)
   - anyám
   - apám

3. **Testvérek** (Брати та сестри)
   - (ваші брати та сестри)

4. **Én** (Я)

5. **Gyerekeim** (Мої діти) - якщо є

**Корисні додаткові слова**:

## Культурна нотатка

::info

**Сім'я в угорській культурі**:

Угорці дуже цінують сімейні зв'язки. Традиційно сімейні зустрічі відбуваються:

- **Vasárnap** (неділя) - сімейний обід
- **Karácsony** (Різдво) - головне сімейне свято
- **Húsvét** (Великдень) - традиційні сімейні традиції

Звернення до старших:

- Дітей часто вчать звертатися до старших на "ön" (формально)
- До бабусь/дідусів - "nagymama", "nagypapa" або пестливо "mamó", "papó"
- До тіток/дядьків завжди "nagynéni", "nagybácsi" (навіть якщо вони не родичі!)

::

::note

**Цікавий факт**:

Слово **nagynéni** (тітка) та **nagybácsi** (дядько) використовується не тільки для родичів! Діти так звертаються до будь-яких дорослих, яких вони знають (наприклад, до сусідів, вчителів тощо). Це знак поваги.

::
