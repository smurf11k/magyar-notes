# Числа від 1 до 100

## Числа від 1 до 10

Ці числа потрібно вивчити ідеально, тому що всі інші числа будуються на їх основі.

| Число | Угорською   | Вимова        |
| ----- | ----------- | ------------- |
| 1     | egy         | [едь]         |
| 2     | kettő / két | [кетте / кет] |
| 3     | három       | [харом]       |
| 4     | négy        | [недь]        |
| 5     | öt          | [ет]          |
| 6     | hat         | [хот]         |
| 7     | hét         | [хет]         |
| 8     | nyolc       | [ньолц]       |
| 9     | kilenc      | [кіленц]      |
| 10    | tíz         | [тіз]         |

::warning

**Важливо про "2"**:

Число "2" має дві форми:

- **kettő** [кетте] - використовується окремо (при рахуванні: egy, kettő, három...)
- **két** [кет] - використовується перед іменниками (két alma = два яблука)

::

---

## Числа від 11 до 19

Ці числа потрібно вивчити як окремі слова:

| Число | Угорською             | Вимова        |
| ----- | --------------------- | ------------- |
| 11    | tizenegy              | [тізенедь]    |
| 12    | tizenkettő / tizenkét | [тізенкетте]  |
| 13    | tizenhárom            | [тізенхаром]  |
| 14    | tizennégy             | [тізеннедь]   |
| 15    | tizenöt               | [тізенет]     |
| 16    | tizenhat              | [тізенхот]    |
| 17    | tizenhét              | [тізенхет]    |
| 18    | tizennyolc            | [тізенньолц]  |
| 19    | tizenkilenc           | [тізенкіленц] |

::tip

**Логіка числівників 11-19**:

Всі числа від 11 до 19 складаються за формулою: **tizen + (число 1-9)**

- tizen + egy = tizenegy (11)
- tizen + három = tizenhárom (13)
- tizen + kilenc = tizenkilenc (19)

::

## Десятки (20, 30, 40...)

| Число | Угорською | Вимова      |
| ----- | --------- | ----------- |
| 10    | tíz       | [тіз]       |
| 20    | húsz      | [хус]       |
| 30    | harminc   | [хормінц]   |
| 40    | negyven   | [недьвен]   |
| 50    | ötven     | [етвен]     |
| 60    | hatvan    | [хотвон]    |
| 70    | hetven    | [хетвен]    |
| 80    | nyolcvan  | [ньолцвон]  |
| 90    | kilencven | [кіленцвен] |
| 100   | száz      | [саз]       |

::note

**Запам'ятайте особливі форми**:

- 20 = **húsz** (не "kétvan"!)
- 30 = **harminc** (не "háromvan"!)
- 40, 50, 60, 70, 80, 90 = закінчуються на **-van/-ven**

::

---

## Числа від 21 до 99

Всі складені числа утворюються за простою формулою: **десятки + число 1-9**

### Приклади:

- 21 = huszonegy [хусонедь]
- 25 = huszonöt [хусонет]
- 32 = harminckettő [хормінцкетте]
- 47 = negyvenhét [недьвенхет]
- 53 = ötvenhárom [етвенхаром]
- 68 = hatvannyolc [хотвонньолц]
- 74 = hetvennégy [хетвеннедь]
- 89 = nyolcvankilenc [ньолцвонкіленц]
- 95 = kilencvenöt [кіленцвенет]

::info

**Правило з'єднання**:

Десятки та одиниці пишуться **разом**, без пробілу та без сполучника (не як в українській "двадцять один").

- 34 = harmincnégy (одне слово!)
- 67 = hatvanhet (одне слово!)

::

## Спеціальні випадки використання

### Вік

**Hány éves vagy?** [хань евеш водь] - Скільки тобі років?

Відповіді:

- **Húsz éves vagyok.** - Мені 20 років.
- **Harminckettő éves vagyok.** - Мені 32 роки.

::tip

**Слово "éves"**:

Після числа завжди додається слово **éves** (річний/літній):

- 18 éves = 18-річний
- 25 éves = 25-річний

::

### Ціни

**Mennyibe kerül?** [меньїбе керюл] - Скільки коштує?

- **Száz forint** - 100 форинтів
- **Ötezer forint** - 5000 форинтів
- **Húszezer forint** - 20000 форинтів

### Час

- **Hány óra van?** [хань ора вон] - Котра година?
- **Öt óra van.** - П'ята година.
- **Tíz óra van.** - Десята година.

## Порядкові числівники (1-10)

Порядкові числівники утворюються додаванням закінчення **-dik/-odik/-edik/-ödik**:

| Кількісний | Порядковий | Переклад  |
| ---------- | ---------- | --------- |
| egy        | első       | перший    |
| kettő      | második    | другий    |
| három      | harmadik   | третій    |
| négy       | negyedik   | четвертий |
| öt         | ötödik     | п'ятий    |
| hat        | hatodik    | шостий    |
| hét        | hetedik    | сьомий    |
| nyolc      | nyolcadik  | восьмий   |
| kilenc     | kilencedik | дев'ятий  |
| tíz        | tizedik    | десятий   |

::warning

**Виняток**:

**Перший** = **első** (не "egydik"!)
**Другий** = **második** (не "kettődik"!)

::

## Практичне використання

### Номери телефонів

Угорські номери телефонів читаються попарно:

**06 30 123 4567** читається:  
nulla-hat, harminc, százhuszonhárom, negyvenöthathetven-hét

### Адреси

**Kossuth utca 23.** [коссут утцо хусонхарем]  
Вулиця Кошута, будинок 23

### Дати

- **1990** = ezerkilencszázkilencven
- **2024** = kétezer-huszonnégy

## Математичні операції

- **plusz** [плус] - плюс (+)
- **mínusz** [мінус] - мінус (-)
- **szorozva** [сорозво] - помножити (×)
- **osztva** [оство] - поділити (÷)
- **egyenlő** [еджелне] - дорівнює (=)

### Приклади:

- **Kettő plusz három egyenlő öt** - 2 + 3 = 5
- **Tíz mínusz négy egyenlő hat** - 10 - 4 = 6
- **Három szorozva négy egyenlő tizenkettő** - 3 × 4 = 12

---

## Корисні фрази з числами

- **Hány?** [хань] - Скільки? (для обчислюваних)
- **Mennyi?** [меньї] - Скільки? (для необчислюваних)
- **Hányan vannak?** [ханьон воннок] - Скільки їх?
- **Hány órakor?** [хань оракор] - О котрій годині?
- **Hány darab?** [хань дороб] - Скільки штук?

### Діалог у магазині:

**A:** Mennyibe kerül ez az alma?  
**B:** Háromszáz forint egy kiló.  
**A:** Akkor kérek két kilót.  
**B:** Az hatszáz forint.

_(Переклад: Скільки коштує це яблуко? - 300 форинтів за кіло. - Тоді прошу два кілограми. - Це 600 форинтів.)_

## Завдання

### Завдання 1: Базові числа

- Вивчити напам'ять числа від 1 до 10
- Вивчити числа від 11 до 19
- Вивчити всі десятки (20, 30, 40... 100)
- Практикувати вимову кожного числа 5 разів

### Завдання 2: Складені числа

- Потренуватися утворювати числа від 21 до 99
- Написати 10 випадкових чисел словами
- Прочитати їх вголос та перевірити

### Завдання 3: Практичне використання

- Навчитися говорити свій вік
- Навчитися говорити свій номер телефону
- Навчитися говорити свою адресу
- Навчитися запитувати ціну

### Завдання 4: Диктант

- Знайти партнера або використати записи
- Написати під диктант 20 чисел
- Перевірити та виправити помилки

## Гра для запам'ятовування

::tip

**Гра "Швидкий рахунок"**:

1. Встановіть таймер на 2 хвилини
2. Рахуйте від 1 до 100 угорською якомога швидше
3. Записуйте свій час
4. Повторюйте щодня, намагаючись побити рекорд

Це чудова вправа для автоматизації числівників!

::

::note

**Щоденна практика**:

- Рахуйте сходинки угорською
- Називайте номери автобусів
- Говорьте ціни в магазині
- Називайте час на годиннику

::

## Числа більше 100 (бонус)

Якщо ви швидко освоїли 1-100, ось структура для більших чисел:

- **200** = kétszáz
- **300** = háromszáz
- **1000** = ezer
- **2000** = kétezer
- **10000** = tízezer
- **100000** = százezer
- **1000000** = millió

### Приклади:

- **2500** = kétezer-ötszáz
- **15000** = tizenötezer
- **250000** = kétszázötvenezer

::info

**На наступний урок**:

Ми вивчимо **дні тижня та місяці**. Це логічне продовження після чисел, тому що нам знадобляться числівники для дат!

::
