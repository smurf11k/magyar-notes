# Кольори та базові прикметники

## Основні кольори

| Українською  | Угорською     | Вимова          |
| ------------ | ------------- | --------------- |
| білий        | fehér         | [фехер]         |
| чорний       | fekete        | [фекете]        |
| червоний     | piros / vörös | [пірош / верош] |
| синій        | kék           | [кек]           |
| жовтий       | sárga         | [шарго]         |
| зелений      | zöld          | [зелд]          |
| помаранчевий | narancs       | [норончш]       |
| фіолетовий   | lila          | [ліло]          |
| рожевий      | rózsaszín     | [рожосін]       |
| коричневий   | barna         | [борно]         |
| сірий        | szürke        | [сюрке]         |

::warning

**Два слова для "червоний"**:

- **piros** - яскраво-червоний (частіше використовується)
- **vörös** - темно-червоний, багряний (більш офіційне)

Для початківців краще використовувати **piros**.

::

---

## Відтінки кольорів

### Світлі відтінки:

- **világos** [вілагош] = світлий
  - világos kék = світло-синій
  - világos zöld = світло-зелений
  - világos barna = світло-коричневий

### Темні відтінки:

- **sötét** [шетет] = темний
  - sötét kék = темно-синій
  - sötét zöld = темно-зелений
  - sötét szürke = темно-сірий

### Яскраві/бліді кольори:

- **élénk** [еленк] = яскравий
  - élénk piros = яскраво-червоний
  - élénk sárga = яскраво-жовтий

- **halvány** [холвань] = блідий
  - halvány rózsaszín = блідо-рожевий
  - halvány kék = блідо-синій

::tip

**Як описати колір**:

Прикметник світлості/темноти ставиться **ПЕРЕД** назвою кольору:

- világos + kék = світло-синій
- sötét + piros = темно-червоний

::

## Прикметники розміру

| Українською | Угорською   | Вимова       | Антонім     |
| ----------- | ----------- | ------------ | ----------- |
| великий     | nagy        | [нодь]       | kicsi / kis |
| маленький   | kicsi / kis | [кічі / кіш] | nagy        |
| довгий      | hosszú      | [хоссу]      | rövid       |
| короткий    | rövid       | [ревід]      | hosszú      |
| широкий     | széles      | [селеш]      | keskeny     |
| вузький     | keskeny     | [кешкень]    | széles      |
| високий     | magas       | [могош]      | alacsony    |
| низький     | alacsony    | [олочонь]    | magas       |
| товстий     | vastag      | [воштог]     | vékony      |
| тонкий      | vékony      | [веконь]     | vastag      |

::note

**kis vs kicsi**:

- **kicsi** - використовується окремо: "Ez kicsi" (Це маленьке)
- **kis** - використовується перед іменником: "kis ház" (маленький будинок)

Це схоже на "kettő" vs "két" з числівників!

::

## Прикметники форми

- **kerek** [керек] - круглий
- **négyzet alakú** [недьзет олоку] - квадратний
- **háromszög alakú** [харомсег олоку] - трикутний
- **ovális** [оваліш] - овальний
- **egyenes** [еджене́ш] - прямий
- **görbe** [гербе] - кривий

## Прикметники якості

### Позитивні:

- **jó** [йо] - хороший, добрий
- **szép** [сеп] - гарний, красивий
- **új** [уй] - новий
- **tiszta** [тісто] - чистий
- **friss** [фріш] - свіжий
- **könnyű** [кеньню] - легкий (по вазі/складності)
- **érdekes** [ердекеш] - цікавий
- **kellemes** [келлемеш] - приємний

### Негативні:

- **rossz** [росс] - поганий
- **csúnya** [чунья] - потворний
- **régi** [реґі] - старий
- **piszkos** [піскош] - брудний
- **nehéz** [нехез] - важкий
- **unalmas** [уноломош] - нудний
- **kellemetlen** [келлемлен] - неприємний

::info

**Пари антонімів**:

- jó ↔ rossz (хороший ↔ поганий)
- szép ↔ csúnya (гарний ↔ потворний)
- új ↔ régi (новий ↔ старий)
- tiszta ↔ piszkos (чистий ↔ брудний)
- könnyű ↔ nehéz (легкий ↔ важкий)

::

## Прикметники температури та відчуттів

- **meleg** [мелег] - теплий, жаркий
- **hideg** [хідег] - холодний
- **forró** [форро] - гарячий
- **langyos** [лонгьош] - теплуватий
- **hűvös** [хювош] - прохолодний

### Смакові відчуття:

- **édes** [едеш] - солодкий
- **sós** [шош] - солоний
- **savanyú** [шовонью] - кислий
- **keserű** [кешерю] - гіркий
- **csípős** [чіпеш] - гострий (пекучий)
- **finom** [фіном] - смачний
- **fincsi** [фінчі] - дуже смачний (розмовне)

## Як використовувати прикметники

### Основне правило:

В угорській мові прикметники **НЕ ЗМІНЮЮТЬСЯ** за родами та числами!

- **piros alma** - червоне яблуко
- **piros almák** - червоні яблука
- **piros autó** - червона машина
- **piros autók** - червоні машини

::tip

**Це легко!**:

На відміну від української, де треба змінювати прикметник (червоний-червона-червоне-червоні), в угорській форма завжди однакова!

::

### Порядок слів:

**Прикметник + Іменник**

- **nagy ház** - великий будинок
- **szép virág** - гарна квітка
- **kék ég** - синє небо

### Декілька прикметників:

- **nagy, szép ház** - великий, гарний будинок
- **kis, piros autó** - маленька, червона машина

---

## Конструкція "щось є яким"

### З дієсловом "van":

**Іменник + прикметник**

- **A ház nagy.** [о хаз нодь] - Будинок великий.
- **Az autó piros.** [оз ауто пірош] - Машина червона.
- **Ez szép.** [ез сеп] - Це гарне.
- **Az jó.** [оз йо] - Те добре.

::note

**Az vs Ez**:

- **Ez** = це (те, що близько)
- **Az** = те (те, що далеко)

Докладніше про це в наступних уроках!

::

## Практичні фрази

### Описуємо речі:

- **Milyen?** [міен] - Який/яка/яке?
- **Milyen színű?** [міен сіню] - Якого кольору?

### Діалог:

**A:** Milyen az autód?  
**B:** Piros és nagy.  
**A:** És milyen a házad?  
**B:** Kicsi, de szép.

_(Переклад: Яка твоя машина? - Червона та велика. А який твій будинок? - Маленький, але гарний.)_

### Опис людини:

- **magas** - високий
- **alacsony** - низький
- **fiatal** - молодий
- **öreg** - старий
- **szép** - гарний/гарна
- **csinos** [чінош] - симпатичний/симпатична

---

## Ступені порівняння (вступ)

### Базова форма:

- **nagy** - великий

### Вищий ступінь:

- **nagyobb** - більший (додається -bb)

### Найвищий ступінь:

- **legnagyobb** - найбільший (leg- + прикметник + -bb)

### Приклади:

- **szép** → **szebb** → **legszebb** (гарний → гарніший → найгарніший)
- **jó** → **jobb** → **legjobb** (хороший → кращий → найкращий)
- **rossz** → **rosszabb** → **legrosszabb** (поганий → гірший → найгірший)

::warning

**Детальніше про ступені порівняння**:

Це лише ознайомлення! Ми докладно вивчимо ступені порівняння в одному з наступних уроків.

::

## Завдання

### Завдання 1: Кольори

- Вивчити всі 11 основних кольорів
- Навчитися називати кольори речей навколо вас
- Описати 10 предметів за кольором
- Вивчити конструкції "світлий/темний колір"

### Завдання 2: Прикметники розміру

- Вивчити всі пари антонімів розміру
- Практикувати з реальними предметами
- Скласти 10 речень з різними прикметниками
- Порівняти 5 пар предметів

### Завдання 3: Опис предметів

- Вибрати 5 предметів у кімнаті
- Описати кожен 3-4 прикметниками
- Практикувати питання "Milyen?"
- Записати описи та прочитати вголос

### Завдання 4: Діалоги

- Скласти діалог про улюблені кольори
- Скласти діалог-опис кімнати
- Скласти діалог про предмети (що це, який він)
- Практикувати вголос

## Гра для запам'ятовування

::tip

**Гра "Я бачу"**:

Граєте з партнером або самостійно:

**A:** "Látok valamit, ami piros." (Я бачу щось червоне)  
**B:** "Ez az alma?" (Це яблуко?)  
**A:** "Igen!" / "Nem!"

Чудова гра для практики кольорів та прикметників!

::

::note

**Щоденна практика**:

Коли бачите предмет, спробуйте описати його угорською:

- Який колір?
- Який розмір?
- Яка форма?
- Який на дотик?

::

## Культурна нотатка

::info

**Кольори в угорській культурі**:

- **Piros-fehér-zöld** (червоний-білий-зелений) - кольори угорського прапора
- **Fehér** (білий) - символ чистоти, часто на весіллях
- **Fekete** (чорний) - траур, офіційність
- **Piros** (червоний) - свята, урочистості

::

::note

**На наступний урок**:

Ми вивчимо **особові займенники та дієслово "to be" (lenni)**. Це дуже важливо для побудови речень!

::
