# Magyar Notes

**Magyar Notes** is a personal Hungarian language learning website built for guided tutoring and long-term self-study.

The site is designed to support structured learning from **A1 to C2**, with clear explanations, lesson notes, and references that students can revisit anytime.

---

## About the Project

- Focused on **Hungarian (magyar) language learning**
- Written primarily for a **Ukrainian-speaking student**, with possible future expansion
- Lessons are written in **MDX** (Markdown + components)
  - Regular `.md` files are also supported
- Content-first approach: explanations, tips, reminders, and learning notes
- Built to be **reusable and extensible** as the curriculum grows

The project currently contains **A1 lessons** and will be expanded gradually up to **C2 level**.

---

## Tech Stack (high-level)

- Static site
- MDX / Markdown for lesson content
- Component-based layouts
- No framework lock-in for lesson writing

(Implementation details may evolve over time.)
